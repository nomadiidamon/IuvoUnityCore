void IuvoUnity.Extensions.MeshRendererExtensions.EnsureDoubleSided ( Mesh mesh)
static

Modifies the given Mesh to render as double-sided by inverting normals and optionally flipping triangle winding.

Parameters
meshThe mesh to modify.

This method inverts the normals and swaps the winding order of triangles to allow both sides of the mesh to be visible. Use this with shaders that respect backface culling settings.

38 // Flip normals for double-sided rendering
39 Vector3[] normals = mesh.normals;

41 {
